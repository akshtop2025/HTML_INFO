HTML Block and Inline Elements

  • Every HTML element has a default display value, depending on what type of element it is.
  • There are two display values: block and inline.

    Block-level Elements

  1. A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
  2. A block-level element always takes up the full width available (stretches out to the left and right as far as it can).
  3. Two commonly used block elements are: <p> and <div>.
  4. The <p> element defines a paragraph in an HTML document.
  5. The<div> element defines a division or a section in an HTML document

    Inline Elements

  1. An inline element does not start on a new line, and the browsers add space (a margin) before and after the element.
  2. An inline element does not take up the full width available (stretches out to the left and right as far as it can).
  3. Two commonly used inline elements are: <span> and <a>.
  4. The <span> element defines a span of text in an HTML document.
  5. The<a> element defines a hyperlink in an HTML document.
