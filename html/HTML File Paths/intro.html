Two Type of Paths

    File paths are used when linking to external files, like:

  • Web pages
  • Images
  • Style sheets
  • JavaScripts
  1. Relative Path
  2. Absolute Path

1. Relative Path

1.Relative Path In the following example, the file path points to a file in the images folder located in the folder one level up from the current folder:

  1. Example:
        syntax  :    <img src="../images/picture.jpg" alt="Mountain">

2. Absolute Path

An absolute file path is the full Path to a file:

  1. Example:
            syntax:  

            <base href="../img/">

            <img src="/Users/user_name/Desktop/folder_name/file_name.png" alt=">
